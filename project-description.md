# Interactive Information VIsualization - D3 Visualization project "Magic Lenses"

## Initial project idea

**Base Visualization**
- the base visualization will be a visualization of time-series data 
- the visualization will display a line graph of the yearly carbon emissions of different countries
- the user can select countries of interest with a checkbox
- a searchbar will help the user to quickly find the country they are looking for
- the time span can be configured as well

**Magic Lens**
- a ChronoLens can be hovered over one or multiple line charts
- the ChronoLens can display configurable insights
- examples: derivatives (ie increase / decrease of emissions), carbon emissions with respect to country size by population, ...
