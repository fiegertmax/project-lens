- the drag-and-drop funtionality of the lens is not explained (both in world map and on line charts) --> a click leads to no response which is counterintuitive
- (strg + scrolling is counterintuitive for zooming lens / increasing size: not implemented as it is intuitive at the laptop which will be used for presentation; issue is for mouse-interaction only)

- country combinations and continents are counterintuitive because the use-case is country-scoped
